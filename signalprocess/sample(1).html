快速傅里叶变换算法

快速傅里叶变换算法利用分治的思想，对于N点的傅里叶变换，先计算N/2点，以此类推。时间复杂度为 。

利用迭代计算傅里叶变换需要三个循环。以8点fft为例，分析如下：首先输入不按顺序输入，需要进行变址运算，之后进入三层循环；首先是有4组，每组1个碟式运算→第二次迭代有2组，每组2个→第三次有1组，每组4个，总共迭代了3次；用r来控制迭代次数，初始r为1，每次更新都乘以2，当r≥N/2时，停止迭代；用p来控制有多少组，组数为(N/2)/r；用q来控制有多少次碟式运算，次数为r；最后W的指数取(q-1)*((N/2)/r)。 

xxxxxxxxxx

                    

                    

                    

                    

                        

                            

                            

                            
function X = FFT(x, type)

                        

                        
    if nargin == 1

                        
        type = 'time';

                        
    end

                        
    

                        
    x = extend(x);

                        
    N = length(x);

                        
    N_2 = N / 2;

                        
    W = exp(-1j*2*pi/N);

                        
    

                        
    mindigits = length(dec2bin(N-1));

                        
    X = zeros(1, N);

                        
    for i=0:N-1

                        
       bin = dec2bin(i, mindigits);

                        
       X(i+1) = x(bin2dec(bin(end:-1:1))+1); 

                        
    end

                        
    

                        
    r = 1;

                        
    while r <= N_2

                        
        k = N/2 / r;

                        
        for p = 0:k-1

                        
            for q = 1:r

                        
                m = 2*p*r+q;

                        
                n = (2*p+1)*r+q;

                        
                [X(m), X(n)] = ...

                        
                    butterfly(X(m), X(n), W^((q-1)*k));

                        
            end

                        
        end

                        
        r = r * 2;

                        
    end

                        
end

function X = IFFT(x, type)

                        

                        
    if nargin == 1

                        
        type = 'freq';

                        
    end

                        
    

                        
    x = extend(x);

                        
    N = length(x);

                        
    N_2 = N / 2;

                        
    W = exp(1j*2*pi/N);

                        
    

                        
    mindigits = length(dec2bin(N-1));

                        
    X = zeros(1, N);

                        
    for i=0:N-1

                        
       bin = dec2bin(i, mindigits);

                        
       X(i+1) = x(bin2dec(bin(end:-1:1))+1); 

                        
    end

                        
    

                        
    r = 1;

                        
    while r <= N_2

                        
        k = N/2 / r;

                        
        for p = 0:k-1

                        
            for q = 1:r

                        
                m = 2*p*r+q;

                        
                n = (2*p+1)*r+q;

                        
                [X(m), X(n)] = ...

                        
                    butterfly(X(m), X(n), W^((q-1)*k));

                        
            end

                        
        end

                        
        r = r * 2;

                        
    end

                        
    X = X / N;

                        
end

周期卷积计算

让长度为 的信号 ，通过长度为 的系统 ，如果通过频域的乘积再逆变换来计算卷积，难以得到正确的结果，因为通过这种方法计算卷积其实是两个周期信号卷积。因此，为了得到正确的结果，需要提高频域采样率来增大周期避免混叠。当 时，即可得到普通卷积的结果。

xxxxxxxxxx

                    

                    

                    

                    

                        

                            

                            

                            
function y = CONV(x1, x2, N)

                        

                        
    if nargin < 3

                        
        N = length(x1) + length(x2) - 1;

                        
    end

                        
    

                        
    y = zeros(1, N);

                        
    

                        
    x1 = extend(x1, N);

                        
    x2 = extend(x2, N);

                        
    

                        
    x2(2:end) = x2(end:-1:2);

                        
    for n=1:N

                        
        y(n) = sum(x1 .* x2);

                        
        tmp = x2(end);

                        
        x2(2:end) = x2(1:end-1);

                        
        x2(1) = tmp;

                        
    end

                        
end

function x = extend(x, N)

                        
    len = length(x);

                        
    if N > len

                        
        tmp = zeros(1, N);

                        
        tmp(1:len) = x;

                        
        x = tmp;

                        
    elseif N < len 

                        
        times = ceil(len/N);

                        
        tmp = zeros(1, times*N);

                        
        tmp(1:len) = x;

                        
        for i=1:times-1

                        
           tmp(1:N) = tmp(1:N) + tmp(i*N+1:(i+1)*N);

                        
        end

                        
        x = tmp(1:N);

                        
    end

                        
end

无限长序列的卷积

有的时候输入信号 会是无限长的，如一段语音信号滤波，事先无法知道信号长度。因此，可以将信号分段通过系统。但是这样会让卷积结果产生误差。为了得到正确的卷积结果，通常有两种方法：重叠相加法和重叠保留法。

重叠相加法：假设LTI系统长度为P，信号分为多个长为L的子信号，输出卷积长度为L+P-1；然而分析可知最后P-1个信号是不完整的，因此这P-1个信号应该与后一个子信号卷积后的前P-1个信号相加才能得到正确的结果。

重叠保留法：假设P<L，那么由循环卷积的定义可得，前P-1个信号是不正确的，因此将前P-1个信号去除，只保留后面的L-(P-1)个信号。

下面是重叠相加法的matlab代码：

xxxxxxxxxx

                    

                    

                    

                    

                        

                            

                            

                            
function y = LTI(x, h, type)

                        

                        
    persistent P;

                        
    persistent L;

                        
    persistent T;

                        
    persistent y_old;

                        
    if nargin == 0

                        
        y = y_old;

                        
        return

                        
    end

                        
    if nargin == 2

                        
        type = 'overlap_add';

                        
    end

                        
    

                        
    if isempty(P)

                        
        P = length(h);

                        
    elseif P ~= length(h)

                        
        error('inpersistent length h');

                        
    end

                        
    if isempty(L)

                        
        L = length(x);

                        
    elseif L ~= length(x)

                        
        error('inpersistent length L');

                        
    end

                        
    if isempty(T)

                        
        T = type;

                        
    elseif T ~= type

                        
        error('inpersistent type');

                        
    end

                        
    

                        
    if isempty(y_old)

                        
        y_old = zeros(1, P-1);

                        
    end

                        
    ytmp = CONV(x, h);

                        
    ytmp(1:P-1) = ytmp(1:P-1) + y_old;

                        
    y = ytmp(1:L);

                        
    y_old = ytmp(L+1:L+P-1);

                        
end

h;

y = [];

                        
for i = 1:length(x)/L

                        
    y = [y, LTI(x((i-1)*L+1:i*L), h)];

                        
end
